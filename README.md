# 🎓 Student Hub Website

## 📌 Overview
Student Hub is a simple multi-page website built using HTML and CSS.  
It is designed to help students access study resources, career advice, and productivity tips in one place.

This project demonstrates my front-end development skills and ability to structure a responsive multi-page website.

---

## 💻 Technologies Used
- HTML5
- CSS3

---

## 📄 Pages Included
- Home Page
- Study Resources Page
- Career Advice Page
- Productivity Tips Page
- Contact Page

---

## ✨ Features
- Multi-page website structure
- Navigation bar across all pages
- Card-based layout design
- Contact form (frontend)
- Clean and simple UI design
- Consistent styling using a single CSS file

---

## 🎯 Purpose of the Project
This project was created to demonstrate my ability to:
- Build structured websites using HTML and CSS
- Design user-friendly layouts
- Create multi-page navigation systems
- Apply basic UI/UX principles

It also forms part of my portfolio as an Information Technology in Business Systems graduate.
